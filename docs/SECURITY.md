# ملاحظات الأمان

- صلاحية الأدمن تُفحص من `ADMIN_IDS` داخل السيرفر.
- Gemini لا يملك أوامر لتعديل قاعدة البيانات أو صلاحيات المستخدمين.
- معالجة تحديثات البوت متسلسلة افتراضيًا لمنع تداخل المحادثات متعددة الخطوات.
- الملفات الكبيرة تُرفض قبل تنزيلها إلى السيرفر.
- الملفات المؤقتة تُحذف بعد الإرسال أو التحليل.
- النسخ الاحتياطية يمكن تشفيرها بواسطة Fernet.
- الأخطاء التقنية لا تظهر للمستخدم؛ راجع Railway Logs.
- لا تخزن بيانات بطاقات أو كلمات مرور بريد داخل هذا البوت.

## ما لا يقدمه هذا الإصدار

- حماية موزعة ضد DDoS.
- استرجاع تلقائي للنسخة الاحتياطية؛ الفحص فقط، لتجنب مسح البيانات بالخطأ.
- تشفير شامل لكل حقول قاعدة البيانات.
- تعدد خوادم متزامن لنفس بوت polling.
