# Study Commander Bot - Manual Settings, Button Ordering, Profile Edit, Habits Patch

هذا الباتش يضيف فقط الميزات المطلوبة بدون تغيير بقية النظام إلا للربط الضروري.

## المضاف

### 1) زر ضبط يدوي
زر جديد في القائمة الرئيسية:

`⚙️ ضبط يدوي`

يفتح شاشة فيها:
- `✏️ تعديل معلوماتي`
- `🔄 تغيير نظامي`
- `🌱 إضافة عادة`
- `📋 عاداتي`

### 2) تعديل معلومات المستخدم
داخل زر `👤 ملفي` تظهر أزرار الضبط، ويستطيع الطالب تعديل:
- الاسم الثلاثي
- الكلية
- المرحلة
- العمر/الطول/الوزن اختياريًا

### 3) تغيير النظام
زر `🔄 تغيير نظامي` يفتح تجربة نظام علمية لمدة 14 يومًا، مثل:
- نظام صباحي
- نظام نوم مبكر
- نظام دراسة ثابت
- نظام مخصص

ويطلب:
- وقت الاستيقاظ
- وقت النوم
- قانون النظام الأساسي

### 4) إضافة عادة
زر `🌱 إضافة عادة` يبني عادة بطريقة علمية:
- اسم العادة
- سببها
- المرساة Habit Anchor
- أقل خطوة Tiny Action
- المكافأة
- مدة التجربة: 7 / 14 / 21 / 30 يوم

### 5) ترتيب الأزرار من لوحة الأدمن
داخل:

`👑 لوحة الأدمن` -> `🧩 الأزرار`

تمت إضافة زر:

`↕️ ترتيب الأزرار`

يسمح بتغيير موقع أي زر داخل شاشته بصيغة:

`صف,عمود`

مثال:

`2,1`

### 6) إصلاح prayer_hint
يتضمن هذا الباتش ملف `app/services/prayer.py` المصحح حتى لا يظهر خطأ:

`ImportError: cannot import name 'prayer_hint'`

