# V6 Prayer Reminders Patch

هذه الإضافة تضيف فقط ميزة أذكار الصلاة الاختيارية + تعديل حفزني ليكون قرآنيًا + إيقاف البومودورو قبل الصلاة بدقيقة.

## الملفات المعدلة/المضافة

- `app/models.py`
- `app/services/prayer.py`
- `app/services/motivation.py`
- `app/handlers/prayer_reminders.py`
- `app/handlers/pomodoro.py`
- `app/keyboards.py`
- `app/services/buttons.py`
- `app/bot.py`
- `assets/motivation/quotes.json`
- `assets/motivation/rsail_min_al_quran.pdf`

## ما الذي تضيفه؟

- زر مستقل: `🕌 أذكار الصلاة`.
- تفعيل/إلغاء تفعيل.
- اختيار المحافظة من محافظات العراق كأزرار.
- تذكير بثلاثة أوقات حسب النمط الشيعي:
  - صلاة الصبح.
  - صلاة الظهر والعصر.
  - صلاة المغرب والعشاء.
- كل تذكير يحتوي آية ورسالة قصيرة تربط الصلاة بالدراسة.
- البومودورو يتوقف قبل الصلاة بدقيقة إذا كانت الصلاة داخل مدة الجلسة.
- زر `🔥 حفزني` صار يستخدم رسائل قرآنية فقط من بنك الاقتباسات المرفق.

## مصدر أوقات الصلاة

الكود يحاول جلب المواقيت من API حقيبة المؤمن/شبكة الكفيل باستخدام الإحداثيات والمحافظة. إذا فشل الاتصال، يستخدم fallback احتياطي حتى لا يتوقف البوت.

## ملاحظة مهمة

إذا عندك قاعدة بيانات قديمة، الجداول الجديدة ستنشأ تلقائيًا عند تشغيل البوت:

- `prayer_settings`
- `prayer_time_cache`
