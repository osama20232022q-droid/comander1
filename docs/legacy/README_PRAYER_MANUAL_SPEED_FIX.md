# Prayer Manual Times + Speed Fix

استبدل هذه الملفات فقط داخل مشروع البوت:

```text
app/models.py
app/services/prayer.py
app/handlers/prayer_reminders.py
app/handlers/pomodoro.py
```

## ماذا يصلح؟

1. يضيف داخل زر **🕌 أذكار الصلاة** خيار:
   - **🕘 تعديل أوقات الصلاة يدويًا**
   - **🔄 استخدام توقيت حقيبة المؤمن**

2. عند الضغط على تعديل الأوقات يدويًا يطلب البوت 3 أوقات بهذا الشكل:

```text
04:12 12:05 18:40
```

والترتيب هو:

```text
الصبح الظهر/العصر المغرب/العشاء
```

3. البومودورو يعتمد الأوقات اليدوية إن وجدت، ويتوقف قبل الصلاة بدقيقة.

4. إذا ألغيت الأوقات اليدوية يرجع البوت يستخدم توقيت حقيبة المؤمن/التوقيت التلقائي.

5. لتسريع البوت:
   - تم تقليل زمن انتظار API الصلاة إلى 2.5 ثانية بدل الانتظار الطويل.
   - تم إضافة memory cache لمواقيت الصلاة اليومية.
   - تم تقليل استعلامات قاعدة البيانات داخل job التذكير.

## متغير اختياري للسرعة

أضف في Railway إذا تريد:

```env
PRAYER_REMOTE_TIMEOUT=2
BUTTON_CACHE_TTL=120
DB_POOL_SIZE=5
DB_MAX_OVERFLOW=10
```

ثم اعمل Redeploy.

## الجداول الجديدة

سيتم إنشاء جدول جديد تلقائيًا:

```text
prayer_manual_times
```

لا يحتاج حذف قاعدة البيانات.
