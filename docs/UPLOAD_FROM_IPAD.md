# الرفع إلى GitHub من الآيباد

الطريقة الأنظف هي تطبيق Working Copy:

1. فك ضغط ملف V8 داخل تطبيق Files.
2. افتح Working Copy وسجّل دخول GitHub.
3. Clone للمستودع.
4. انقل **محتويات** مجلد V8 إلى جذر المستودع داخل Working Copy.
5. تأكد أن `Dockerfile` و`run.py` ظاهرين مباشرة، وليس داخل مجلد إضافي.
6. Commit ثم Push.
7. لا ترفع `.env`.

بعد الـ Push، Railway يعيد النشر تلقائيًا إذا كان المستودع مربوطًا به.
